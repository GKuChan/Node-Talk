Node-Talk（Node.js我要说）
=============

An open guide for using Node.js

Node.js从环境搭建，到基础入门，再到设计实现的小教程，旨在帮助新手入门、帮助新手实现以Node为主力开发环境、提升Node在大学生圈内的学习氛围。
